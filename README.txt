---------------------------Sonic Generations Mod--------------------------
--------------------------E-Rank Generations v1.0-------------------------

This is a mod for Sonic Generations that replace S-rank with E-rank, including HUD and result animation.

------------------------------------------------------------
Mod usage
------------------------------------------------------------
-This mod works ONLY with HedgeModManager, SonicGMI is NOT supported!
-If you are using Unleashed HUD or Unleashed Model mod, this mod must be HIGHER PRIORITY than them, otherwise it will not show E-rank texture or crash the game
-Right-click mod to configure what Sonic model you are using, if this is not done correctly, Sonic's E-rank animation may appear broken
